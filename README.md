## 1. The Diffusion Revolt runs Locally

A complete guide to running LLaDA, an 8B masked diffusion language model locally on Apple Silicon using MLX. The code implementation is under progress which covers the theory, architecture, and the non-obvious implementation details behind the custom sampling loop, quantization configs, and semi-autoregressive block mode. A different path to language generation, worth understanding.

Read here : https://diffusiononmlx.netlify.app/
